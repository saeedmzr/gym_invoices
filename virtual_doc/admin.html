Admin

Authentication

You can use a simple artisan command to create an admin user php artisan admin:create

In your .env file, you can change admin credentials via ADMIN_EMAIL, ADMIN_PASSWORD, and ADMIN_NAME



Admin Webservices

  • Users : There's a crud control with users that you can control your users (CREATE, READ, UPDATE, DELETE).admin/users
  • Clubs : There's a crud control for your clubs.admin/clubs
  • Memberships :There's also a crud control with the membership of users.admin/memberships
  • Invoices : You can see invoices and specific user invoices.admin/invoices


Note: For more details, you can check the postman collection
